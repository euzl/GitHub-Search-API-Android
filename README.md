# GitHub-Search-API-Android
GitHub API를 활용하여 리포지터리 검색

<br>

## 작업 상황
- Material Design Text Fields `EditText`
- GitHub Search API 통신 `retrofit2`
  - 대기 중 `progressBar` 표시
- 검색 결과 `recyclerView`로 표시
